# Document Header Template
#
# Use this fragment at the top of every generated engineering document.
# Replace all placeholders with real values - no "[Insert here]" or "TBD" allowed.

| Field | Value |
|---|---|
| **Document ID** | `<SKILL-CODE>-<YYYYMMDD>` (e.g., `TSD-20260712`) |
| **Version** | `1.0` |
| **Status** | `Draft` / `In Review` / `Approved` / `Superseded` |
| **Project** | `<Project or Product Name>` |
| **Author(s)** | `<Name, Role>` |
| **Date Created** | `<YYYY-MM-DD>` |
| **Last Updated** | `<YYYY-MM-DD>` |
| **Reviewers** | `<Name, Role>` |
| **Approver** | `<Name, Role>` |

---

> **Status guidance:**
> - `Draft` - work in progress, not yet reviewed
> - `In Review` - shared with reviewers, pending feedback
> - `Approved` - sign-off received, baseline locked
> - `Superseded` - replaced by a newer version (link to new doc)
