# Privacy Policy

**Last Updated:** July 17, 2026
**Effective Date:** July 17, 2026

## Overview

The `engineering-docs` plugin ("the Plugin") is an open-source tool that provides documentation generation skills for AI coding agents. This privacy policy explains how the Plugin handles data.

## Data Collection

**The Plugin does not collect any user data.**

Specifically, the Plugin:
- ❌ Does not collect personal information
- ❌ Does not collect usage analytics
- ❌ Does not collect telemetry data
- ❌ Does not track user behavior
- ❌ Does not collect file contents or project data
- ❌ Does not require user accounts or registration

## Data Processing

**The Plugin processes all data locally on the user's machine.**

When the Plugin generates documentation:
- All processing happens locally within the AI agent's environment
- No data is transmitted to external servers
- No data is stored beyond the local filesystem
- Generated documents remain on the user's machine

## Network Access

**The Plugin does not require network access for core functionality.**

- Skills are markdown files loaded locally by the AI agent
- Scripts execute locally on the user's machine
- The MCP server (`bin/validate.js`) runs locally and reads files only
- Hooks (`hooks/check-progress.js`) run locally and read status only

**Exception:** The AI agent platform (e.g., Claude Code, Gemini CLI) may require network access to communicate with its AI model provider. This is handled by the agent platform, not the Plugin.

## Data Storage

**The Plugin does not store any data beyond what the user creates.**

- Generated documents are saved to the user's local filesystem
- No data is stored in Plugin-specific databases or caches
- No cookies or local storage are used
- No credentials or tokens are stored by the Plugin

## Third-Party Services

**The Plugin does not integrate with third-party services.**

- No analytics services (no Google Analytics, Mixpanel, etc.)
- No error tracking services (no Sentry, Bugsnag, etc.)
- No cloud storage services
- No external APIs

**Note:** The AI agent platform may integrate with third-party services (e.g., AI model providers). This is governed by the platform's own privacy policy, not this Plugin's.

## User Rights

Since the Plugin does not collect any data, there is no user data to:
- Access
- Rectify
- Erase
- Port
- Restrict processing of
- Object to processing of

Users have full control over all files generated by the Plugin, as they reside on the user's local filesystem.

## Children's Privacy

The Plugin does not knowingly collect data from children under 13. As the Plugin does not collect any data, it complies with COPPA and similar regulations.
